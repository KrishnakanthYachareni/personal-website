2. Install the Gatsby CLI


3. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install --force
   ```

4. Install dependencies

   ```sh
   yarn
   ```

5. Start the development server

   ```sh
   npm start
   ```
